# MyOS — Build Plan v2 (the re-cut)
*July 22, 2026 · supersedes the original build plan. Built against three locked artifacts: Feature List v1.1, UX Foundation v1 (incl. the Deck + four D's), and the Warm Glass design kit. The Milestone-0 skeleton (Vite + React + Supabase auth) already runs locally.*

## Standing laws (in DECISIONS.md from day one)

- All of TRUE's engineering law: optimistic writes, three-tier prompt caching, signature-gated judgment caching, RLS immutability, row-checks, length caps, injection framing, honest cursors, one lockfile.
- **The L5 countermeasure, now formal:** when a milestone stalls, the rule is *finish or formally cut* — never start the adjacent fun thing. A cut gets a dated DECISIONS entry.
- Keep the raw (Q2): originals archived beside every distillate.
- No scores, no streaks, no notifications, no proactivity (Q3) anywhere in V1.
- Design law = the kit: tonal wells, no borders/rings (focus = glow), marker A (dot + mono tag), one accent moment per screen, undo everywhere / confirm only for the permanent, the two-register voice, motion timing table.
- Project colors: user-choosable, random default, any hex.
- Day boundary: user's local timezone.

GitHub repo + Vercel deploy of what already runs, then the Warm Glass tokens land as the real CSS foundation (palettes, type, wells, motion), plus the shell bones: the Void with quotes, the lens rail, the composer. **Done = the calm empty app at a real URL, looking like the kit.**

## M3 — One door + the Deck + People
The private ingest address (raw kept + archived, distillation bar, promises extracted); Plaud AutoFlow and reMarkable convert-to-text flowing in; chat-box capture routing with the inline "context or conversation?" ask; the routing learning loop. **The Deck ships here**: cards from every source, the four D's (→Do ↑Delay ←Delegate ↓Delete), swipes on mobile / arrows on desktop, hold-for-finer-moves, everything-in-to-start intake. ## M4 — Now + the day's rhythm
The Now strip (marker-A chips, the ≤12-word WHY layer with signature caching, GONE/FULL/SLIM states, never scored); "what should I do right now" answered decisively in chat; the Day Closer (evening reMarkable page or one-line sign-off seeds tomorrow); reminders that resurface without notifying; local-tz app-day. **Done = the full daily loop — morning glance is right, evening close takes one line.**

## M5 — The record (invisible until summoned)
The immutable ledger + the self-portrait (Identity / Principles / Blindspots / Leverage — one system per Q4) synthesized in the background with the evidence bar; "show me my blindspots" / attention-with-zeros on demand in chat; the Friday impact check as a Friday conversation; TRUE's data imported as clearly-labeled seed memory (fresh start, no amnesia). **Done = Friday's check cites real receipts from your real weeks.**
